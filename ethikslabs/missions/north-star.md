# North Star — EthiksLabs Mission

*Source text for all PRAXIS company missions. Each 360 company inherits from this and focuses it to their domain.*

---

## The goal

John Coates never manually logs into any system, ever, unless absolutely required by a human-only security boundary (biometric, physical hardware key).

Everything else — deployment, configuration, auth setup, DNS, cloud infrastructure, secret management, API calls, monitoring, updates — is handled by the system.

John's role: natural language instruction → final authority → approval.
The system's role: everything between those two points.

---

## The operating principle

CTFB — Compress The Friction Between thought and execution. Read intent. Act. Report.

---

## The infrastructure philosophy

Autoscale by default. Never think about scale. Every infrastructure choice is made so that growth from 1 user to 1,000,000 users requires no architectural changes, no manual intervention — only a pricing tier upgrade.
